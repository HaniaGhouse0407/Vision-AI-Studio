## 🧠 Overview

A unified computer vision platform exposing four tasks through a clean Gradio interface: object detection (YOLOv8, 80 COCO classes), lane detection (Canny + Hough), multi-algorithm edge detection, and K-Means image segmentation.

---

## ✨ Features

- ✅ YOLOv8 object detection — 80 COCO classes, configurable confidence threshold
- ✅ Real-time lane detection with Canny + Hough Transform + ROI masking
- ✅ Edge detection: Canny, Sobel, Laplacian algorithms
- ✅ K-Means image segmentation with adjustable cluster count
- ✅ Clean tabbed Gradio UI — shareable via public link
- ✅ Drop-in real YOLOv8 by uncommenting 3 lines

## 🚀 Quick Start

```bash
# 1. Clone
git clone https://github.com/HaniaGhouse0407/Vision-AI-Studio.git
cd Vision-AI-Studio

# 2. Install dependencies
pip install -r requirements.txt

# 3. Set environment variables (if needed)
cp .env.example .env
# Edit .env with your API keys

# 4. Run
python app.py
```
